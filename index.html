Setting Up Google Apps Script

Step 1: Open Google Sheets → Create new sheet named TDP_Planner_Data


Step 2: In the sheet, go to Extensions → Apps Script


Step 3: Replace all code with the Apps Script code (download button below)


Step 4: Click Deploy → New Deployment → Type: Web App → Execute as: Me → Who has access: Anyone


Step 5: Copy the Web App URL and paste it above


⚠️ Each time you change the script, create a New Deployment (not update) to get a fresh URL.
